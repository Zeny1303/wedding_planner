# Wedding Planner Website

A colorful and interactive wedding planning website inspired by WedMeGood.  
The project includes a vibrant homepage, vendor search, budgeting dashboard, and wedding stories collage.

## Features
- Elegant landing page with vendor and city filters  
- Separate planning dashboard for managing vendors and budgets  
- Interactive budget calculator  
- Beautiful collage page for wedding stories with editable captions and uploads  
- Fully responsive, built with HTML, CSS, and JavaScript
